기본상태
 - 왼쪽 위에 탈출 진행도 -> tab이나 alt + 클릭으로 미션창이 열린다.
   -> 얻은 아이템과 미션 진행도에 따라서 가장 높은 순으로 나열이 된다.
   -> 가장 진행도가 높은 미션의 진행도를 맨 위에 탈출 진행도 옆에 표시한다.

 - 오른쪽 아래 1,2,3 -> 아이템 창 -> 아이템 없으면 걍 비워두기
 - 오른쪽 아래 신분증 -> 플레이어 얼굴 정면에서 카메라 잡아서 띄워주기
      ( 플레이어 상태 애니메이션 표현할 예정 )
 - 오른쪽 아래 경고등 -> 보안레벨 : 1(초록), 2(노란), 3(빨간)
 - 오른쪽 아래 아이템창 위에 아이콘 -> 현재 진행되는 이벤트 (전기나갔다, 경비병 집합상태,,,,)

미니맵(M) -> 스크롤을 통해 다른 층 미니맵보기
 - 미니맵을 켰을 때 본인이 있는 위치의 미니맵이 먼저 화면에 나오게
 - 플레이어 아이콘 (플레이어 상태 추가)
 - 경비병 아이콘 (초록 : 순찰 / 노랑 : 수색 / 빨강 : 추격)
 - CCTV 아이콘 (초록 : 감시X / 주황 : 감시O / 빨강 : 추격on) - 10-15초 간격으로 일정한 패턴으로 바뀌게
